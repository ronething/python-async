## 并发 并行

并发 一个**时间段**内 有几段程序在**同一个** cpu 上运行 但任意时刻只有一个程序在 cpu 上运行

并行 任意时间点上，有**多个**程序同时运行在**多个** cpu 上

## 同步 异步

同步 代码调用 IO 操作时，必须等待 IO 操作完成才返回的调用方式

异步 代码调用 IO 操作时，不必等待 IO 操作完成才返回的调用方式 

## 阻塞 非阻塞

阻塞 调用函数时候当前线程被挂起

非阻塞 调用函数时候当前线程不会被挂起，而是立即返回
